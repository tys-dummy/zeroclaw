## Edit config

```bash
sudo nano /var/lib/docker/volumes/zeroclaw_zeroclaw-data/_data/.zeroclaw/config.toml
```

```bash
sudo chmod 600 /var/lib/docker/volumes/zeroclaw_zeroclaw-data/_data/.zeroclaw/config.toml
```

# Add skills

```bash
sudo cp -r <skill-folder> /var/lib/docker/volumes/zeroclaw_zeroclaw-data/_data/workspace/skills/
```
